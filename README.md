# Spam Message Detection

This project builds a machine learning model to classify text messages as spam or ham (not spam). It demonstrates a complete natural language processing workflow including text preprocessing, feature extraction, model training, evaluation, and prediction.

## Project Overview

Spam detection is a common text classification problem in natural language processing. The goal of this project is to identify whether a given message is spam or legitimate based on its text content.

This project demonstrates practical skills in:

- Text preprocessing
- Feature extraction
- Machine learning model training
- Classification evaluation
- Prediction on unseen messages

## Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Workflow

1. Load and explore the dataset
2. Clean and preprocess text data
3. Convert text into numerical features
4. Train a classification model
5. Evaluate performance using classification metrics
6. Predict spam vs ham messages

## How to Run

1. Install dependencies
2. Launch Jupyter Notebook

```bash
jupyter notebook
```
